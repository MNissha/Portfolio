UI/UX Design Competition Entry - Carlo Rino E-Commerce

A modern e-commerce platform design concept

Project Overview

Competition Brief

  • Design a modern e-commerce platform for Carlo Rino
  • Focus on user experience and brand identity
  • Create a responsive and intuitive interface
  • Incorporate current e-commerce best practices

Design Approach

  • Conducted user research and competitive analysis
  • Created user personas and journey maps
  • Developed wireframes and high-fidelity prototypes
  • Implemented responsive design principles

Tools & Technologies

  • Figma - UI/UX design, prototyping, and collaboration
  • FigJam - User flow diagrams and wireframing
  • Figma Plugins:
    • Auto Layout - For responsive design
    • Content Reel - For quick content population
    • Stark - For accessibility checking
    • Icon8 - For icon resources

Prototype Demo

Core Features

Homepage

Main homepage featuring a clean, modern design with prominent Carlo Rino branding and easy-to-navigate menu structure.

Category section showcasing different product categories with intuitive visual navigation and clear categorization.

Newsletter subscription module with elegant design, offering users exclusive updates and promotional offers.

Comprehensive footer section containing important links, company information, and social media connections.

User Authentication

User-friendly login interface with options for email/password login and Google authentication integration.

Streamlined sign-up form with clear input fields and validation for new user registration.

Shopping Experience

Advanced filtering system allowing users to refine product searches by multiple criteria including price, category, and features.

Personalized wishlist interface where users can save and manage their favorite items for future purchase.

Intuitive shopping cart display showing selected items, quantities, and total cost with easy modification options.

Detailed product page featuring high-quality images, comprehensive product information, and size selection options.

Customer review section displaying user feedback, ratings, and authentic product reviews with photos.

Customer Support

Real-time chat interface providing instant customer support with professional service representatives.

Comprehensive contact form for customer inquiries and feedback submission with multiple department options.

Integrated social media links and feeds keeping customers connected with Carlo Rino's latest updates and promotions.
